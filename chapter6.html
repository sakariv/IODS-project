The original RATS dataset has data on the effect of three treatments on the weight of rats. The data has been taken in a time of 64 days. The measurements for weight for each day of scaling were in their own column. In the modified RATSL dataset the measurements have been put into a single column, with the day of observation as a new variable ‘Time’.

T-test and ANOVA of RATS

PBRS data had two groups. Now with multiple groups the two-sample t-test gives an error, however can we use ANOVA (oneway.test) for multiple groups.

# ANOVA
oneway.test(mean ~ Group, data = RATSL8S1, var.equal = TRUE)
## 
##  One-way analysis of means
## 
## data:  mean and Group
## F = 501.81, num df = 2, denom df = 12, p-value = 2.721e-12
library(dplyr)
library(tidyr)

# for baseline we need the original RATS data
RATS <- read.csv("https://raw.githubusercontent.com/KimmoVehkalahti/MABS/master/Examples/data/rats.txt", header = T, sep = '\t') # separator tab

# Add the baseline from the original data as a new variable to the summary data
RATSL8S2 <- RATSL8S %>%
